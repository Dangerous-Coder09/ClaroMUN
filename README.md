# ClaroMUN Website

This repository contains the official website for the ClaroMUN (Claret Model United Nations) conference. The website is designed to provide information to delegates, advisors, and guests about the conference, including details about the committees, the secretariat, and registration.

## Features

*   **Homepage:** A welcoming introduction to ClaroMUN with quick links to essential information.
*   **Committee Pages:** Detailed information about each committee, including UNSC, WHO, and UNHRC.
*   **Secretariat Information:** Introductions to the student leaders of the conference.
*   **Registration Link:** Easy access to the delegate registration form.
*   **Responsive Design:** The website is designed to be accessible on various devices, including desktops and mobile phones.
*   **Engaging Animations:** Subtle animations to improve user experience.

## 🔧 How to Use

To view the website locally, simply open the `index.html` file in your web browser.

1.  Clone the repository:
    ```bash
    git clone https://github.com/Dangerous-Coder09/Claro_MUN.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Claro_MUN
    ```
3.  Open `index.html` in your favorite browser.
